修改Apache+php上传文件大小限制

发表于 2018-01-20 | 更新于: 2018-01-23 | 阅读次数:
字数统计: 207 | 阅读时长 ≈ 1

1.找到Apache中的php.ini文件

一般是安装目录的bin\apache\apache2.4.23\bin\php.ini，例如，我的电脑中此文件的目录在D:\wamp\bin\apache\apache2.4.23\bin\php.ini

找到php.ini文件后，需要对其进行一些修改！

2.修改php.ini的哪些地方？

  1.max_execution_time = 1200，Maximum execution time of each script, in seconds 脚本最大1执行时间，单位是秒

  2、max_input_time = 1000 接收传输数据的最长时间，单位也是秒

  3、memory_limit = 128M 一个脚本占用内存的上限

  4、post_max_size = 128M post方式传输的数据最大的上限

  5、file_uploads = On 允许文件上传

  6、upload_max_filesize = 100M 上传的文件上限

  7、max_file_uploads = 20 一次允许上传20个文件

记住，一定是修改Apache中的php.ini文件，修改php中的php.ini文件是不起作用的
