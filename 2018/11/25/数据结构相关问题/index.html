2. 判断是否是二叉搜索树

5. 递归测试S是否是回文串

9. 搜索带索引的二叉搜索树中的第K个小的关键码

假设一颗带索引的二叉搜索树，root 指向其根节点，树中每个结点具有如下形式：

Lsize left data right
该结点左子树中的结点个数加一 左子树 int型 右子树

试用 Java 语言写出一个递归的 findk 函数，即搜索这棵带索引的二叉搜索树中第K个小的关键码结点。
分析：Lsize表示该结点左子树中的结点个数加1，即该表示该结点按从小到大排列的位置下标，二叉搜索树的特点：首先是一颗二叉排序树，并且结点的左子树都小于该点，右子树都大于该点的值。
