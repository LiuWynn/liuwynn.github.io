注意事项：

  • 使用循环时，对列表中的每一个元素，都将执行循环指定的步骤（如栗子里的，print(magician)），而不管列表包含多少个元素。如果列表包含一百万个元素，Python 就重复执行指定的步骤一百万次，且通常执行的速度非常快，不信你可以试试。
  • 编写 for 循环时，对于用于存储列表中每个值的临时变量（如栗子里的，magician），可随意取名。但我们建议取有意义的名称，别人看代码的时候好理解。
