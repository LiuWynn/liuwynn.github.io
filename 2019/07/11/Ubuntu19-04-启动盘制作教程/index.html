Ubuntu19.04 启动盘制作教程

发表于 2019-07-11 | 更新于: 2019-07-12 | 阅读次数:
字数统计: 1.2k | 阅读时长 ≈ 4

前言

最近几天翻出了以前的旧电脑，心痒难耐，决定捣鼓 Linux 系统。对比了 Linux 各个系统以后，决定安装 Ubuntu 桌面版体验一下。

制作系统盘

因为我是在 windows 系统中制作的系统盘，这里就只说明 windows 系统下制作 Ubuntu 系统盘的方法。

1.1 下载官方镜像

可以去官方网站下载，这里给个 传送门

如果你要长期使用 Ubuntu 系统，建议安装 LTS 版本，LTS 版本会得到官方长期免费的安全更新维护支持。

因为我个人是为了尝鲜，所以就选了最新发布的 Ubuntu19.04 系统，仅有 9 个月的安全更新维护支持。

不管你选择的是什么版本的 Ubuntu 系统，制作系统盘的方法都是一样的，本教程都适用。

1.2 制作启动盘

我在 Windows 平台试用了两款 启动盘制作工具：UltraISO 和 Universal USB Installer，下面我将分别讲解一下这两款软件的使用方法。

1.2.1 UtlraISO 的使用

首先下载好 UtlraISO 这里给个 传送门

UtlraISO 是款收费软件，不过下载安装好以后给我们试用的机会，所以如果你是装机的话足够了。

下载的过程不赘诉，傻瓜式 next，直到完成安装。

安装好以后打开软件，打开如图1-1所示：

图1-1

点击 继续使用 按钮，就可以进入软件使用界面了。

软件使用界面如图1-2所示：

图1-2

点击 文件 -> 打开，选择 ISO 镜像文件，然后打开，就看到了如图1-3所示：

图1-3

可以看到 ISO 镜像文件已经被加载进来了。

这个时候，就可以选择 U盘，然后往 U盘里写入此镜像文件了，具体做法如图1-4 所示：


图1-4

点击 启动 -> 写入硬盘映像，然后会弹出一个如图1-5的界面：


图1-5

  • 硬盘驱动器：一般默认即可，除非你电脑同时插入了 2 个及以上 U盘，那么你得选择一下要写入哪个 U盘。

  • 映像文件：没法选，但你要看一下这个映像文件是不是我们要安装那个 ISO，如果不是，你就返回重新打开正确的 ISO 文件即可。

  • 写入方式：网上其他装机的都说 USB-HDD+ 和 USB-HDD 都可以，可能是我个人电脑配置有所区别，在用这两种方式制作好U盘以后，装机时会出现黑屏情况下，左上角光标不断闪烁的现象，然后迟迟无法进入装机界面，所以建议各位小朋友们：先以 USB-HDD+ 或 USB-HDD 方式写入U盘，一般情况下是没问题的，要是出现了和我一样的异常情况，再改用 RAW 方式写入U盘。

选项都确定没问题以后可以点击 写入 按钮，此时会提示你会格式化U盘，我们当然选择 yes，然后静静的等待刻录完成即可，速度根据每个人的电脑写入速度有所区别。

最后但很重要的一点：要写入引导程序，具体做法如图1-6所示：


图1-6

点击 便捷启动 -> 写入新的驱动器引导扇区 -> Syslinux

写入引导程序的速度非常快。

做好这一步，你的 Ubuntu系统 启动盘就已经做好了，开心吧？！

1.2.2 Universal USB Installer 的使用

Universal USB Installer 无需安装，下载即可用，而且是免费的，很激动吧，哈哈。

按照惯例，给出 传送门

软件很小，也就 1.7MB 左右

软件界面打开如图1-7所示（你打开各个选项是处于未选择的状态，和我的有点不一样，接下来会讲解）：


图1-7

  • Step1: 选择要安装的系统名，比如我要安装的是 Ubuntu系统，所以选择 Ubuntu即可。

  • Step2： 选择镜像文件，点击 browser按钮，选择要写入的 ISO 镜像即可。

  • Step3: 选择硬盘驱动器，大白话就是选择你要写入的 U盘

  • 最后：勾选 will Fat32 Format

然后点击 Create即可。

等待写入完成，Ubuntu系统启动盘就制作好啦。

结尾语

有时候闲来无事动手捣鼓捣鼓 Linux 系统还挺有意思的，而且现在社会发展越来越趋向于电子化，不管是不是计算机相关专业的，对电脑使用的越熟练也越有优势。
